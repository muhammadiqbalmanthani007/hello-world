Abbreviation Syntax

Emmet uses syntax similar to CSS selectors for describing elements' positions inside genrated tree and element's attribute

Element

You can use elements's names like div or p to generate HTML tags. Emmet doesn't have a predefined set of available tag names, you can write any word and transform it to a tag: div → <div> </div> , foo → <foo> </foo> and so on.

Nesting Operator

Nesting Operators are used to position abbreviation elements inside generated tree: wheter it should be placed inside or near the context element.

Child: >

You can use > operator to nest elements inside each other:
div>ul>li
...will produce 
<div>
     <ul>
        <li>
     <ul>
<div>
